# NodeLab Solutions — Sitio estático (HTML + CSS)

Descripción
- Sitio corporativo estático creado solo con HTML y CSS, pensado para presentar los servicios de NodeLab Solutions.
- Diseño limpio, profesional y corporativo. Iconos monocromáticos, fondo en degradado azul-gris suave.

Archivos principales
- `index.html` — Página principal con secciones: Inicio, Nuestros servicios, Por qué elegirnos, Implementación, Contacto.
- `formacion-personalizada.html` — Página de servicio: Formación Personalizada.
- `soluciones-preconfiguradas.html` — Página de servicio: Soluciones Preconfiguradas.
- `implementacion-mantenimiento.html` — Página de servicio: Implementación Integral y Mantenimiento.
- `agentes-autonomos.html` — Página de servicio: Agentes Autónomos.
- `styles.css` — Estilos compartidos para todas las páginas.

Cómo ver el sitio localmente
1. Abrir un terminal en la carpeta del proyecto (`/workspaces/codespaces-blank`).
2. Ejecutar un servidor estático, por ejemplo:

```bash
python3 -m http.server 8000
```

3. Abrir en el navegador: http://localhost:8000/index.html

Notas y consideraciones
- Todos los textos en las páginas se han incluido tal como se solicitaron (no se han resumido ni modificado).
- Navegación: en `index.html` los enlaces del menú usan anclas (`#inicio`, `#servicios`, `#porque`, `#implementacion`, `#contacto`). En las subpáginas el menú vuelve a `index.html#...`.
- El proyecto no usa JavaScript ni frameworks externos: solo HTML y CSS.
